# Hybrid-UAV-UGV — 陆空两栖机器人

## 项目简介

本仓库为《智能系统创新实践》课程项目的统一管理仓库。项目目标是设计并制作一台陆空两栖机器人，可在特定场景下完成特定任务，兼具实用性与创新性。

## 目录结构

```
Hybrid-UAV-UGV/
├── docs/           # 所有文档交付物（方案简报、周报、中期报告、总结报告、演示材料）
├── research/       # 技术调研与文献（论文、竞品分析、技术调研报告）
├── design/         # 方案设计（系统架构、机械/电气/软件方案、控制策略）
├── hardware/       # 硬件实现文件（3D模型、PCB、BOM、装配说明）
├── software/       # 软件代码（机载代码、地面站、通信、仿真）
├── tests/          # 测试与验证（单元测试、集成测试、实地测试数据）
├── tools/          # 辅助工具（标定工具、通用脚本）
├── resources/      # 参考资料（数据手册、教程、报告模板）
└── assets/         # 媒体素材（图片、视频、海报、Logo）
```

详细目录说明请参见各目录下的 README 文件。

## 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 目录 | 全小写 + kebab-case | `flight-controller/` |
| 代码文件 | 全小写 + 下划线 | `pid_controller.py` |
| 文档文件 | 语义化 + 日期前缀 | `2026-07-01-week1-report.md` |
| 设计文件 | 组件名 + 版本号 | `chassis-v2.step` |
